## Scripts Disponíveis

No diretório do projeto, você pode executar:


Para executar o servidor Backend:

### `cd server`
### `npm run dev`

Para executar o projeto Frontend:

### `cd web`
### `npm run dev`

Para executar o projeto Mobile:

### `cd mobile`
### `npx expo start`
